### 介绍
 给用户发红包，自己可以获得订单6%的提成。

###功能介绍
1.领取美团和饿了么红包
2.分销功能，用户推荐给其他人，还能拿到下级的返利
3.公众号推送，每天提醒用户点餐
4.返利提现，订单的返利，用户可以直接提现到微信钱包
5.推广海报生成，用户可以生成自己的专属海报，实现分销，赚取佣金

### 软件架构
前端uniapp、后端PHP+vue
